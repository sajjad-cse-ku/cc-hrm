| Command                      | Description                                     | Example                                       |
| ---------------------------- | ----------------------------------------------- | --------------------------------------------- |
| `module:make`                | Create a new module                             | `php artisan module:make Blog`                |
| `module:make` *(multiple)*   | Create multiple modules at once                 | `php artisan module:make Blog User Auth`      |
| `module:use`                 | Set a module as the default context             | `php artisan module:use Blog`                 |
| `module:list`                | List all available modules                      | `php artisan module:list`                     |
| `module:migrate`             | Run migrations for a module (or all)            | `php artisan module:migrate Blog`             |
| `module:migrate-rollback`    | Rollback migrations for a module (or all)       | `php artisan module:migrate-rollback Blog`    |
| `module:migrate-refresh`     | Refresh (rollback + migrate) module migrations  | `php artisan module:migrate-refresh Blog`     |
| `module:migrate-reset`       | Reset all migrations for a module (or all)      | `php artisan module:migrate-reset Blog`       |
| `module:seed`                | Run database seeder for a module (or all)       | `php artisan module:seed Blog`                |
| `module:publish-migration`   | Publish migration files for a module (or all)   | `php artisan module:publish-migration Blog`   |
| `module:publish-config`      | Publish config files for a module (or all)      | `php artisan module:publish-config Blog`      |
| `module:publish-translation` | Publish translation files for a module (or all) | `php artisan module:publish-translation Blog` |
| `module:enable`              | Enable a module                                 | `php artisan module:enable Blog`              |
| `module:disable`             | Disable a module                                | `php artisan module:disable Blog`             |
| `module:update`              | Update a module                                 | `php artisan module:update Blog`              |
